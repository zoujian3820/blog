## linux命令

### **sudo** 以管理员身份(root)来执行 命令

### **df** 检查文件系统的磁盘占用情况

```
df      用KB来做单位显示使用情况
df -TH  用M来做单位显示使用情况
```

### **cd /** 进根目录

### **rm**   删除文件

### **rm -rf**  删除目录

### **mv node-v10.15.3-linux-x64 nodejs**   重命名

### **ls** 查看当前目录结构
```
ls -a   列出所有文件和目录,包含隐藏文件
ls -la  详细列出所有目录结构,包含隐藏文件
```

### **top** 查看 cpu/内存/进程 使用情况
- 按 m 键 看使用百分比
- Ctrl + c 退出 top 命令

### **nmcli d** 查看安装在本机的网卡

### **nmtui** 打开网络管理员

### **systemctl restart network** 重启网络服务
