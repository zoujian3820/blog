# Problem: JS 是多线程么?绑定一个事件的回调函数是宏任务还是微任务?

*[interview]: start
js是单线程的，但它可通过worker开启子线程，主线程是只会有一个的，事件的回调属于宏任务，当事件被触发时回调会被放入到宏任务队列中，并等待执行
*[interview]: end
