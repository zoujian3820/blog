# Problem: 如何优雅的调试需要发布的 Npm 包？

\*[interview]: start

- 在需要调试的 npm 包目录下结构下的控制台输入 npm link 这个命令会把当前包映射到本地的一个全局的 npm 包里面；
- 在引用的目录结构下的控制台输入 npm link 包名称 这个命令会把本地引用的这个 npm 包的路径定位到全局的 npm 包下；
- 全局的 npm 包相当于一个中转站，在编辑区域与引用区域之间中转。

\*[interview]: end
